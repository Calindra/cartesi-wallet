# CARTES-WALLET

This is a typescript based Wallet implementation for Cartesi Dapps to handle different types of Assets.

## Methods

```
Wallet {
      balance_get,
      _ether_deposit,
      _erc20_deposit,
      _erc721_deposit,
      ether_withdraw,
      ether_transfer,
      erc20_withdraw,
      erc20_transfer,
      erc721_withdraw,
      erc721_transfer
    }
```
